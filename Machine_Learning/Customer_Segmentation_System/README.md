# Customer Segmentation System

## Project Overview

This project performs customer segmentation using RFM (Recency, Frequency, Monetary) analysis and K-Means clustering on retail transaction data. PCA is used to visualize customer groups, helping businesses identify meaningful customer segments for marketing and retention strategies.

---

## Problem Statement

Analyze retail customer purchasing behavior and group customers into meaningful segments based on their transaction history.

---

## Dataset

- **Source:** UCI Online Retail Dataset (Kaggle)
- **Records:** ~541,000 transactions
- **Customers:** 4,000+
- **Features:** 8

## Key Insights

- Customers can be grouped based on purchase recency, frequency, and spending.
- High-value customers contribute significantly to overall revenue.
- At-risk customers can be targeted with retention campaigns.
- PCA provides a clear visualization of customer clusters.
