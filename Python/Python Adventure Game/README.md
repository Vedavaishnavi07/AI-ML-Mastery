# Python Adventure Game

## Overview

The Lost Temple Adventure is a text-based adventure game built in Python using Object-Oriented Programming.

The player explores different locations, collects items, fights enemies, and unlocks the Final Temple to defeat the Ancient Dragon.

## Features

- Player Class
- Enemy Class
- Inventory System
- Health System
- Combat System
- Gold Collection
- Ancient Key Quest
- Final Dragon Boss
- Multiple Exploration Paths
- Random Events

## Technologies

- Python 3
- Object-Oriented Programming
- Random Module

## How to Run

```bash
python main.py
```

- Classes and Objects
- Functions
- Loops

- Lists
