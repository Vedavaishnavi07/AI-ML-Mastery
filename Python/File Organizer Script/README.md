# File Organizer Script

## Overview

This Python script automatically organizes files inside a folder based on their file extensions.

The script creates category folders (Images, Documents, Videos, Music, Archives, Code, Others) and moves each file into the appropriate folder.

## Features

- Automatically creates folders
- Organizes files by extension
- Supports multiple file categories
- Handles duplicate filenames
- Skips hidden files
- Ignores existing directories
- Displays a summary after execution
- Uses error handling for safe file operations

## Technologies Used

- Python 3
- pathlib
- shutil
- File I/O

## How to Run

```bash
python organizer.py
```

Enter the folder path when prompted.

Example:

```
C:\Users\YourName\Downloads
```
