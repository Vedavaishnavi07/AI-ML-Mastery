# Customer Orders Analyzer

## Overview

A Python-based data analysis project that reads customer order data from a CSV file and generates useful business insights.

## Features

- Load customer order data from CSV
- Calculate total revenue
- Analyze top products by revenue
- Identify top customers
- Calculate category-wise revenue
- Generate formatted analysis report

## Technologies Used

- Python
- CSV module
- Dictionaries
- Functions
- Sorting algorithms
- File handling

## Project Structure
Customer Orders Analyzer
│
├── main.py
├── orders.csv
├── report.txt
└── README.md

## Output

The program generates:

- Total number of orders
- Total revenue
- Top 5 products
- Top 5 customers
- Category revenue breakdown
- Detailed report.txt file

## How to Run

1. Open terminal inside the project folder

2. Run: python main.py
