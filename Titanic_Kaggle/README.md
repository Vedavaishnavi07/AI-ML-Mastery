# Titanic Survival Prediction using Machine Learning

## Overview
This project predicts whether a passenger survived the Titanic disaster using supervised machine learning. It was developed as part of the Kaggle Titanic competition.

## Dataset
- Source: Kaggle Titanic - Machine Learning from Disaster
- Training Data: 891 passengers
- Test Data: 418 passengers

## Project Workflow

1. Data Loading
2. Exploratory Data Analysis (EDA)
3. Missing Value Handling
4. Feature Engineering
5. Data Encoding
6. Model Training
7. Model Evaluation
8. Kaggle Submission

## Models Used

- Logistic Regression
- Decision Tree
- Random Forest

## Model Performance

| Model | Validation Accuracy |
|--------|--------------------:|
| Logistic Regression | 79.89% |
| Decision Tree | 74.30% |
| Random Forest | 83.24% |

## Technologies Used

- Python
- Pandas
- NumPy
- Scikit-learn
- Matplotlib

## Output

A `submission.csv` file was generated and successfully submitted to the Kaggle Titanic competition.
