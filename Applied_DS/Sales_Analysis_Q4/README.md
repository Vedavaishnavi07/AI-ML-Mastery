# Sales Analysis - Regional Q4

## Project Overview

This project analyzes regional sales data using the Superstore dataset. The objective is to identify top-performing regions, profitable product categories, and high-revenue states through Exploratory Data Analysis (EDA) and data visualization.

---

## Dataset

- Dataset: Sample Superstore
- Records: 9,994
- Columns: 13

---

## Technologies Used

- Python

---

## Analysis Performed

- Data Loading
- Dataset Exploration
- Statistical Summary
- Business Overview
- GroupBy Analysis
- Data Visualization

---

## Visualizations

1. Sales by Region
2. Profit by Region
3. Sales by Category
4. Profit by Category
5. Top 10 States by Sales
6. Sales vs Profit

---

## Business Insights

- West region generated the highest sales.
- West region generated the highest profit.
- Technology is the highest revenue-generating category.
- Furniture has high sales but comparatively low profit.
- California contributes the highest sales among all states.
