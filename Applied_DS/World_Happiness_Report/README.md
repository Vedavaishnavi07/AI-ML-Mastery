# World Happiness Report Analysis

## Project Overview

This project analyzes the World Happiness Report (2015–2023) to understand the factors that influence happiness across countries.

## Technologies Used

- Python
- Pandas
- NumPy
- Matplotlib

## Dataset

World Happiness Report (2005–2025)

Filtered for analysis:
**2015–2023**

## Analysis Performed

- Data Cleaning
- Correlation Analysis
- Linear Regression
- Country Ranking
- Time-Series Analysis
- Data Visualization

## Visualizations

- Top 10 Happiest Countries
- Happiness Trend (2015–2023)
- GDP vs Happiness
- Social Support vs Happiness
- Freedom vs Happiness
- Corruption vs Happiness

## Key Findings

- Social support is the strongest driver of happiness.
- GDP has a strong positive relationship with happiness.
- Finland ranks highest in the latest analysis.
- Healthy life expectancy contributes significantly.
- Generosity has the weakest relationship with happiness.
