# COVID-19 Data Analysis

## Project Overview

This project analyzes global COVID-19 data using Python and Pandas. It performs data cleaning, exploratory data analysis (EDA), statistical summaries, and visualizations to understand the spread and impact of COVID-19 across different countries.

---

## Dataset

Source:
Our World in Data (OWID)

Dataset:
owid-covid-data.csv

---

## Technologies Used

- Python
- Pandas
- Matplotlib
- NumPy

---

## Project Features

- Data Loading
- Data Cleaning
- Missing Value Analysis
- Duplicate Removal
- Statistical Summary
- Top 10 Countries by Cases
- Top 10 Countries by Deaths
- Continent-wise Analysis
- GDP Analysis
- Life Expectancy Analysis
- Global COVID Summary
- Data Visualization

---

## Visualizations

The project generates the following charts:

- Cases by Continent
- Top 10 Countries by Cases
- Top 10 Countries by Deaths
- GDP vs Total Cases
- Life Expectancy vs Total Deaths
- Worldwide Daily New Cases

All graphs are automatically saved inside the **images/** folder.
